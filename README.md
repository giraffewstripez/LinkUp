# LinkUp
CISC 4900 Project:
LinkUp Mission Statement
LinkUp’s mission is to increase awareness of community events taking place throughout Brooklyn and Manhattan during Summer 2026. We define community events as opportunities for New Yorkers to connect with like-minded neighbors through shared interests, experiences, and cultural engagement.
Because this initiative is ambitious and cannot be fully developed within a five-week timeframe, our initial focus is on highlighting events that serve specific communities before expanding our coverage more broadly.
